# 텔레그램 발주봇 사용 매뉴얼 (가맹점용)

i'S ORDER 텔레그램 발주봇을 처음 쓰는 가맹점을 위한 안내서입니다. 등록부터
발주, 도매처 관리까지 실제 봇이 응답하는 문구 그대로 정리했습니다.

---

## 1. 시작하기 — 가맹점 등록

1. 대표님께 안내받은 텔레그램 봇을 검색해서 채팅방을 엽니다.
2. 아무 메시지나 한 번 보냅니다 (예: "안녕하세요").
3. 봇이 자동으로 등록 절차를 시작합니다. 순서대로 답장해주세요.

| 순서 | 봇이 묻는 것 | 예시 답장 |
|---|---|---|
| 1 | 지점명 | 오산세교점 |
| 2 | 연락처(전화번호) | 010-1234-5678 |
| 3 | 사업자등록번호 | 123-45-67890 |

3단계까지 입력하면 아래 메시지가 옵니다.

> 등록 신청이 완료되었습니다. 대표님 승인을 기다려주세요.
> (내 chat_id: 123456789)

이 시점부터는 **대표님이 승인할 때까지 다른 명령은 사용할 수 없습니다.**
아무 메시지를 보내도 "승인을 기다리는 중"이라는 안내만 돌아옵니다.

- **승인되면**: 별도 알림 없이 바로 모든 기능이 열립니다. 아무 메시지나 다시
  보내보면 정상 작동하는지 확인할 수 있습니다.
- **반려되면**: 반려 사유가 담긴 메시지가 바로 전송됩니다. 사유를 확인하고
  대표님께 직접 문의해주세요.

---

## 2. 도매처 계정 등록 (필수)

발주를 담으려면 **본인이 실제로 쓰는 도매처 아이디/비밀번호**를 먼저
등록해야 합니다. 등록 안 된 도매처는 가격비교에는 나오지만 실제 담기는
실패합니다.

**시작 명령어**: `계정등록` (또는 `도매처등록`, `도매처계정등록`)

봇이 순서대로 물어봅니다:

1. 등록할 도매처 이름 → **야미몰 / 과자생각 / 삼봉몰 / 현동몰 / 무마켓 / 또요몰** 중 하나 입력
2. 해당 도매처 아이디 입력
3. 해당 도매처 비밀번호 입력

완료되면 "OOO 계정이 등록되었습니다." 메시지가 오고, 다른 도매처도
등록하려면 `계정등록`을 다시 보내면 됩니다. **여러 도매처를 각각 등록해두는
걸 권장합니다** — 등록해둔 도매처가 많을수록 품절 시 자동으로 다른
도매처로 넘어갈 수 있는 폭이 넓어집니다.

---

## 3. 발주하기 (핵심 기능)

### 3-1. 상품 목록 보내기

사고 싶은 상품명을 **한 줄에 하나씩** 보내면 됩니다. 여러 개면 줄바꿈으로
구분해서 한 번에 보내세요.

```
하리보 골드베렌
몬스터에너지 망고
코카콜라 제로
```

**수량 지정**: 상품명 뒤에 숫자를 붙이면 그 개수만큼 담습니다 (안 붙이면 1개).

```
하리보 골드베렌 4
몬스터에너지 망고 2
```

### 3-2. 상품이 여러 개로 잡히면

"하리보"처럼 여러 상품이 검색되면 번호로 골라달라고 먼저 물어봅니다.

- 하나만 고르기: `2`
- 여러 개 고르기(쉼표로 구분): `2,4`
- 해당하는 상품이 없으면: `스킵`
- 전체 취소: `취소`

### 3-3. 가격비교 결과 확인

모든 상품이 정리되면 아래처럼 결과가 옵니다.

```
아래 상품으로 최저가 담기를 준비했습니다:

• 하리보 골드베렌 x4개 → 야미몰 14,000원
• 몬스터에너지 망고 x2개 → 과자생각 4,600원

담을까요? '확인'이라고 답장해주세요. 취소하려면 '취소'라고 답장해주세요.
```

- **담기 실행**: `확인` (또는 `네`, `예`, `ok`, `담아줘` 등)
- **취소**: `취소` (또는 `아니`, `no`, `cancel`)

찾지 못한 상품이 있으면 "찾지 못한 상품 (직접 확인 필요)" 목록으로 따로
안내됩니다 — 도매처 사이트에서 직접 검색해보세요.

### 3-4. 담기 결과 확인

`확인` 후 담기가 진행되면 "장바구니에 담는 중입니다..." 안내가 오고,
완료되면 상품별 성공/실패 결과가 옵니다.

```
담기 결과:

✓ 하리보 골드베렌 - 야미몰 담기 완료
✓ 몬스터에너지 망고 - 과자생각 담기 완료
```

### 3-5. 품절 상품 자동/수동 대체

선택된 도매처에서 품절이면 봇이 자동으로 처리를 시도합니다:

1. **1차**: 이번 발주에 이미 포함된 다른 도매처로 자동 재시도 (성공하면
   결과 메시지에 "※최초 선택하신 OOO이(가) 품절이라 다른 도매처로 자동
   변경됨"이라고 안내되니, 버그가 아니라 정상 동작입니다).
2. **2차**: 그래도 안 되면 등록해둔 다른 도매처 후보를 번호로 골라달라고
   따로 물어봅니다.

```
'하리보 골드베렌'은(는) 선택된 도매처에서 품절이에요. 다른 도매처에서 담을까요? 번호로 답장해주세요:

1. 과자생각 14,000원
2. 삼봉몰 14,500원

담지 않으려면 '스킵'이라고 답장해주세요.
```

## 4. 주거래처 설정 (선택)

가격이 동률일 때 우선적으로 담을 도매처를 지정할 수 있습니다.

| 명령어 | 동작 |
|---|---|
| `주거래처 설정 야미몰` | 야미몰을 주거래처로 지정 |
| `주거래처 확인` | 현재 설정된 주거래처 확인 |
| `주거래처 해제` | 설정 해제 |

---

## 5. 도매처 켜기/끄기 (선택)

특정 도매처를 가격비교 대상에서 빼고 싶을 때 사용합니다 (예: 그 도매처
계정이 없거나 당분간 안 쓸 때).

| 명령어 | 동작 |
|---|---|
| `도매처 활성화 삼봉몰` | 삼봉몰을 가격비교에 포함 |
| `도매처 비활성화 삼봉몰` | 삼봉몰을 가격비교에서 제외 |
| `도매처 목록` | 현재 켜짐/꺼짐 상태 전체 확인 |

---

## 6. 도움말

`도움말` (또는 `명령어`, `help`)이라고 보내면 언제든 요약 안내를 다시 볼 수
있습니다.

---

## 7. 전체 명령어 요약

| 명령어 | 설명 |
|---|---|
| (상품명, 줄바꿈으로 여러 개) | 발주 목록 전송 → 가격비교 |
| `확인` / `네` / `예` / `ok` / `담아줘` | 비교 결과 담기 확정 |
| `취소` / `아니` / `no` / `cancel` | 진행 중인 요청 취소 |
| `스킵` | 모호한 상품/품절 대체 선택 시 건너뛰기 |
| `계정등록` | 도매처 아이디/비밀번호 등록 시작 |
| `주거래처 설정 (도매처명)` | 동률 시 우선 도매처 지정 |
| `주거래처 확인` | 현재 주거래처 확인 |
| `주거래처 해제` | 주거래처 설정 해제 |
| `도매처 활성화 (도매처명)` | 해당 도매처를 가격비교에 포함 |
| `도매처 비활성화 (도매처명)` | 해당 도매처를 가격비교에서 제외 |
| `도매처 목록` | 도매처별 켜짐/꺼짐 상태 확인 |
| `도움말` | 명령어 요약 다시 보기 |

지원 도매처: **야미몰 / 과자생각 / 삼봉몰 / 현동몰 / 무마켓 / 또요몰**

## 8. 자주 묻는 상황

**Q. "계정 미등록" 이라고 담기가 실패해요.**
A. 그 도매처 계정을 아직 등록 안 한 상태입니다. `계정등록`으로 먼저
등록해주세요.

**Q. 상품이 "찾지 못한 상품"으로 나와요.**
A. 도매처 카탈로그에서 이름이 다르게 등록돼있거나, 아직 캐시에 없는
상품일 수 있습니다. 상품명을 더 구체적으로 바꿔서 다시 보내보거나,
도매처 사이트에서 직접 확인해주세요.

**Q. 최저가로 골랐는데 다른 도매처로 담겼어요.**
A. 버그가 아닙니다 — 처음 고른 도매처가 품절이라 같은 발주에 포함된 다른
도매처로 자동 전환된 겁니다. 담기 결과 메시지에 안내 문구가 같이 옵니다.

**Q. 등록했는데 계속 "승인을 기다리는 중"이라고 나와요.**
A. 대표님이 아직 승인하지 않은 상태입니다. 대표님께 등록 사실을
알려주세요(메시지에 있는 chat_id를 같이 전달하면 확인이 빠릅니다).
